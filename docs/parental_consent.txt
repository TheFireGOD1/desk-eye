================================================================================
                              CONSENT STATEMENT
================================================================================

Parental/guardian consent required for minors: This app accesses your webcam
to analyze eye features (blink rate and eye openness) for the purpose of
detecting eye strain. No raw video is stored or uploaded by default. If you
are under 18, you must have a parent or guardian review and sign this consent
form before testing.

================================================================================
                           APPLICATION INFORMATION
================================================================================

Application Name: DeskEye
Purpose: Educational tool to monitor and reduce digital eye strain
Developer: Student team (Samsung Solve for Tomorrow project)
Version: 1.0.0

================================================================================
                              WHAT THE APP DOES
================================================================================

DeskEye is designed to help students maintain healthy eyes while using
computers. The application:

1. Accesses the device's built-in webcam
2. Detects facial landmarks to locate the eyes
3. Analyzes eye features including:
   - Blink rate (how often you blink)
   - Eye openness (Eye Aspect Ratio)
   - Time since last break
4. Calculates an eye strain probability score
5. Provides visual and audio reminders to take breaks
6. Stores aggregated health metrics locally on the device

================================================================================
                              DATA COLLECTION
================================================================================

WHAT IS COLLECTED:
- Aggregated eye health metrics (blink rate, strain scores)
- Timestamps of measurements
- Break events (when breaks are taken)
- User preferences and settings

WHAT IS NOT COLLECTED:
- Raw webcam video or images (unless explicitly enabled by user)
- Personal identifying information
- Location data
- Browsing history or other application data

DATA STORAGE:
- All data is stored LOCALLY on the device
- No data is uploaded to any server or cloud service
- Data can be exported to CSV format
- Data can be deleted at any time by uninstalling the app

================================================================================
                              PRIVACY PROTECTION
================================================================================

DeskEye is designed with privacy as a core principle:

✓ All video processing happens locally on the device
✓ No raw video is saved by default
✓ No data is transmitted over the internet
✓ No third-party tracking or analytics
✓ User can disable monitoring at any time
✓ User can delete all data at any time

Optional Feature (DISABLED by default):
- "Save Raw Images" can be enabled in Settings for debugging
- If enabled, webcam frames are saved to local storage
- This feature requires explicit opt-in and shows a warning
- Saved images can be automatically deleted after a set period

================================================================================
                              RISKS AND BENEFITS
================================================================================

POTENTIAL RISKS:
- Minimal risk, similar to using any webcam application
- Possible false positives or false negatives in strain detection
- App is NOT a medical device and does not replace professional eye care
- Should not be solely relied upon for health decisions

POTENTIAL BENEFITS:
- May help reduce digital eye strain symptoms
- Encourages healthy screen time habits
- Provides awareness of eye health during computer use
- Educational tool for understanding eye strain
